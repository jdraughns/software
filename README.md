# Build science online tutorials

1.一键安装命令

```
wget -N --no-check-certificate https://softs.wtf/Bash/ssr.sh && chmod +x ssr.sh && bash ssr.sh
```
或

```
wget -N --no-check-certificate https://softs.wtf/Bash/ssrmu.sh && chmod +x ssrmu.sh && bash ssrmu.sh
```

2.运行脚本命令

```
bash ssr.sh
```

或

```
bash ssrmu.sh
```

### BBR搭建
1.查询内核命令

```
uname -r
```

2.执行安装命令

```
wget -N --no-check-certificate https://softs.wtf/Bash/bbr.sh && chmod +x bbr.sh && bash bbr.sh
```
### 启动BBR
```
bash bbr.sh start
```
### 关闭BBR
```
bash bbr.sh stop
```
### 查看BBR状态
```
bash bbr.sh status
```
